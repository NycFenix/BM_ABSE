# Blockchain Multi-Authority Based Security Encryption (BM-ABSE)

This is a prorotype implementation of the security encryption system described in this [article](https://www.researchgate.net/publication/395764617_A_Blockchain-Enabled_Multi-Authority_Secure_IoT_Data-Sharing_Scheme_with_Attribute-Based_Searchable_Encryption_for_Intelligent_Systems) as a final work of the IOT course from Universade Federal do Rio de Janeiro (course took in first semester of 2026)

It describes a blockchain based data-sharing system that relegates the simple Central Authority (CA) responsible for validating atributted in a simple Atribute-Based Encryption for bettering security in private IOT environments like Healthcare and Military IOT.
